# Proyecto: 15 Páginas Web Enlazadas

## 🧩 Descripción

Este proyecto consiste en la creación de 55 páginas web estáticas desarrolladas únicamente con HTML y CSS. Todas las páginas están enlazadas entre sí, formando un conjunto de interfaces visuales diseñadas como práctica intensiva de maquetación.

No incluye funcionalidades dinámicas ni interacción con JavaScript, ya que el enfoque principal fue fortalecer habilidades de estructura, diseño y organización visual.

---

## 🎯 Objetivo del Proyecto

El propósito de este trabajo fue mejorar la capacidad de:

- Diseñar interfaces limpias y organizadas.
- Estructurar contenido correctamente con HTML.
- Aplicar estilos coherentes y bien distribuidos con CSS.
- Desarrollar disciplina y constancia en proyectos largos.
- Practicar la repetición como método de perfeccionamiento.

Este proyecto representa entrenamiento técnico y constancia en el desarrollo front-end.

---

## 🛠 Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- Enlaces internos entre documentos

---

## 📚 Enfoque de Trabajo

Para completar las 55 páginas, se trabajó bajo una metodología de:

- Reutilización de estructuras base.
- Variación de estilos, colores y distribución.
- Mejora progresiva en cada nueva interfaz.
- Optimización del tiempo de desarrollo.

Cada página explora diferentes composiciones como:

- Secciones principales tipo landing.
- Sistemas de navegación.
- Distribución en columnas.
- Uso de tarjetas (cards).
- Secciones informativas.
- Diseños minimalistas y modernos.

---

## 📈 Aprendizajes Obtenidos

Durante el desarrollo de este proyecto se fortalecieron habilidades como:

✔ Organización del código.  
✔ Consistencia en el diseño visual.  
✔ Manejo eficiente de CSS.  
✔ Control del espaciado y jerarquía visual.  
✔ Velocidad de maquetación.
